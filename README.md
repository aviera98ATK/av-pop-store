# AV Pop Store

_This is an E-Commerce of Funko Pop's, using React, Firebase and Materialize. This proyect is for Codehouse course._

### Pre-requirements 📋

_AV Pop Store depends on several softwares to be installed_

```
NodeJS
```

### Installation 🔧

_First of all we need to install firebase 7.2.3 with --save and --force, due to several errors that i could't solve._

```
npm install firebase@7.2.3 --save --force
```

_After that, we can do the normal installation._

```
npm install
```


## Usage Example

_To run this app is needed to initialize with npm start_

```
npm start
```
